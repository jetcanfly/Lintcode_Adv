### Follow Up 常见方式

一维转二维
- 可以套相同的思路试一试
  - Find Peak Element I/II
  - Trapping Water I/II
  - Subarray Sum/Submatrix Sum

数组变成循环数组
- 循环数组小技巧
 - Continuous Subarray Sum

题目条件加强
- 可能题目的解题方法会变化
 - Wiggle Sort I/II

换马甲(变一个描述，本质不变)
- 本质不变
 - Number of airplane on the Sky/ Meeting Room
 - BackPack Problem
 - Two Sum/ Three Sum/ Four Sum

描述完全不一样，但是方法相同
- 这种题目得去分析
 - 前向型指针的题目
 - Quick Sort/ Bolt Nuts Problem
